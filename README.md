## Installation
Make sure [jax](https://github.com/google/jax#installation) and [jaxtyping](https://github.com/google/jaxtyping) are installed.
In the root folder, run
```shell
pip install -e .
```

## Quickstart

```bash
python scripts/simulate_batch.py
python scripts/plot_batch.py
```
